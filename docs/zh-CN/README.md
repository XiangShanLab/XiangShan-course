## 🎯 课程目标

本课程旨在系统性地教授香山高性能处理器的设计与开发，帮助学习者：

1. 掌握香山处理器的完整开发流程
2. 理解RISC-V架构与香山微架构设计
3. 学会使用香山敏捷开发工具链
4. 具备香山处理器调试与优化能力

## 🎓 三个学习层次

### 第一层：基础掌握（模块1-3）

- **目标**：开芯院新员工、零基础学习者
- **重点**：环境搭建、Chisel基础、RISC-V指令集
- **产出**：能运行简单香山程序

### 第二层：核心精通（模块4-6）（持续产出教学资料中）

- **目标**：合作企业/高校工程师、进阶学习者
- **重点**：微架构优化、性能分析、调试技巧
- **产出**：能参与香山核心开发与优化

### 第三层：高级应用（模块7-10）（后续计划）

- **目标**：香山爱好者、研究人员、工具开发者
- **重点**：模拟器、验证方法、完整工具链
- **产出**：能独立开发香山相关工具和扩展

## 📅 学习建议

1. **按顺序学习**：建议从模块1开始，循序渐进
2. **理论与实践结合**：每个模块都包含实验环节
3. **参与社区**：遇到问题及时在GitHub提问
4. **完成认证**：学完全部课程可准备香山认证考试
